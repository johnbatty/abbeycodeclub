PICO-8: First steps

Before we start to write any code, you need to learn how to find your way around the PICO-8 screens.

Start the PICO-8 application

PICO-8 should already be installed on your computer. Click on the PICO-8 entry in the Start Menu to run it. You will see this screen and hear a little tune when it starts up…

To run a program, you type “RUN” and press the “Enter” key. At the moment this does absolutely nothing, as we haven’t written a program yet!

Go ahead and try it!

We can tell PICO-8 to do stuff by typing at this command prompt. For example, PICO-8 can print messages and do sums!

Again, try it out! If type something the computer it won’t catch fire! Make up your own messages to print!

Computers are really fussy about what you type. If you make just a single mistake, the computer will give you an error message and refuse to run the command or program. If you see an error then you need to check your code and work out what is wrong.

Look at these examples and see if you can spot my mistakes!

You will see these errors when writing code - we all make mistakes! So don’t panic when you get them - the computer is just trying to help you get your code right!

Switch between screens

If you press the “ESC” key (top left key on your keyboard) then you switch between the console window and the editor windows. You use the editor windows to write your code and design your graphics and sound.

Press ESC once and you will see the code editor… We’ll write some code in here very soon!

Press ESC again and you will go back to the console window. Then press ESC again to go back to the code editor!

See the icons in the top right of the code editor? Click on those to see the other editors…
