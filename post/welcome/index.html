Welcome!

Welcome to Abbey Code Club!

I’m very excited about our brand new code club - the first ever code club at Abbey School. I hope you are too!

There’s not much to read here at the moment, but I’ll be updating the website regularly so check back often.

Code clubs will run every Tuesday afternoon 3:30-4:30 in the computer room.

To get started, here are some links to websites that you might find interesting!

Code Club

Home of all Code Clubs. There are nearly 5,000 clubs in the UK!

Lots of projects here, some of which we will be doing in our Code Club. Most of these projects need a passcode to access them. If you are keen to try them out outside of Code Club then tell me and I’ll give you the required code.

Code.org

An American organisation, whose mission is to teach children to code. Sounds like a good place for us to go!

There are some great “Hour of Code” games here that are a fantastic way to get started with programming, including Minecraft, Star Wars, and Frozen.

Can you complete them all?!

You should also watch some of the videos - they have a lot of famous people in them, including Bill Gates (who started Microsoft), Mark Zuckerberg (who started Facebook) and Barack Obama (the USA president!). All of these people think it is really important for children to learn to code!

Scratch

The most popular way to write programs using graphical blocks. A great tool to start with. It means you don’t have to type very much, and teaches you the main principles of computer programming.

There are instructions on the right hand side of the page to help you learn how to use it!

Raspberry Pi

Everyone’s favourite low-cost computer. And it is British!

Have you got one?! If so, tell me what you do with it…

BBC micro:bit

A low cost “wearable” computer, recently released by the BBC (yes, the people who make TV programmes!). Check out the video.

Through the generosity of Code Club, we will shoortly be getting four of these, so I look forward to us using them later in the term!

If you want to get some practice in, you can follow a tutorial, write programs and run them on a “virtual” micro:bit here. Tip: it will easiest if you start off using the Block Editor!

Pimoroni

A British company who started off making cases for Raspberry Pis, but have now grown into making their own electronics, and selling all sorts of add-ons.

Designers and makers of the fantastic Flotilla electronics kit. I’ve got one of these, and we’ll do something with it later in the year!

Adafruit

American designer and maker of all sorts of cool electronic components.

Loads of great projects!

SparkFun

Another American maker of all sorts of cool electronic components! They have a large number of tutorials. Most of them are quite advanced electronics, but they might give you some ideas!

Make:

A very established American website that covers making all sorts of things - not just computer-related ones!

Lots of super-creative ideas here…!
